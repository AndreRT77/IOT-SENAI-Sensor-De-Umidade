# IOT-SENAI-Sensor-De-Umidade

Neste Projeto fizemos um sensor de umidade, utilizamos um LDR, Esp32, Sensor de Umidade, resistores e Leds, fizemos a simulação e o código no Wowki em formato C++, e depois, utilizamos o Arduino IDE para mandar para o Modelo físico.
